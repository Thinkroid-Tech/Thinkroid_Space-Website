Why Space

AI should be for everyone

Today's AI agent tools are built for developers — command lines, config files, prompt engineering. That leaves most people behind. We believe the benefits of AI agents shouldn't be locked behind a technical barrier.

Thinkroid Space uses a gamified pixel-art interface to turn complex multi-agent orchestration into something visual and intuitive. You see your agents walk between desks, collaborate at meeting tables, and present at whiteboards. If you can play a game, you can manage an AI team.

Our goal is simple: anyone — regardless of age, background, or technical skill — should be able to build an AI workforce, automate their work, and participate in the AI era. Space is the interface that makes that possible.

The Problem

AI tools today are fragmented

Complex Setup

Traditional multi-agent frameworks require deep technical knowledge, custom code, and hours of configuration.

No Visibility

Text-based interfaces hide what your agents are doing. You can't see collaboration, status, or bottlenecks.

Isolated Tools

Chat assistants are single-agent. You need a team of AI workers, not just one assistant.

Features

Everything your AI office needs

Gamified Pixel-Art Interface

Watch your AI employees walk between desks, collaborate at meeting tables, and present at whiteboards. Not a gimmick — it's how you understand complex agent systems at a glance.

Multi-Agent Orchestration

Create teams of specialized agents. A project manager delegates to researchers, writers, and analysts. Each agent has their own role, skills, and personality.

Persistent Memory

Two-tier memory system with forgetting curves. Agents remember past conversations, learn from mistakes, and build context over sessions.

Governance & Approvals

Built-in governance loop with 12 capabilities: budget tracking, performance reviews, conflict resolution, and more. You stay in control.

100% Local Deployment

Runs entirely on your machine via Docker. Your conversations, memories, and outputs never leave your infrastructure. Zero cloud dependency.

Any LLM Support

Bring your own API key. Works with Claude, GPT-4, Gemini, Llama 3, Mistral, DeepSeek, and any OpenAI-compatible API including local models via Ollama.

AI-Guided Onboarding

Create agents through a 7-step wizard with Athena guiding each decision. She suggests names, personas, skills, and models — then fills the form for you. Interactive option cards let you choose with a click.

Workspace File Manager

Browse, preview, upload, and download workspace files directly in the browser. Syntax-highlighted code preview, image viewing, folder ZIP downloads, and glob/regex search.

Docker Container Management

Agents can spin up sandbox containers to run code safely, deploy persistent services like web servers and databases, and manage the full container lifecycle — all isolated per agent with resource limits.

Cron Scheduler

Schedule recurring tasks, governance checks, and scripts with persistent cron jobs. Agents can create their own schedules, and you manage everything from a unified panel.

Built-in AI Assistant

Athena — Your AI Guide

A system-level assistant that lives inside your workspace. Press Cmd+/ to summon her from any screen.

  • Context-Aware — Knows which panel you're viewing and adapts her answers accordingly
  • Live System Access — Queries agents, tasks, settings, and agent memories in real time
  • Inline Approvals — Create tasks, send messages, edit files, and run commands with approve/deny buttons right in the chat
  • 4 Permission Modes — Plan (read-only), Ask (manual approval), Auto Edit (files auto-approved), Full Auto (everything auto)
  • Rich Markdown — Tables, syntax-highlighted code blocks, and formatted output rendered natively
  • Configurable — Custom prompts, model selection, per-tool approval modes, and conversation compaction
  • Interactive Option Cards — Presents clickable single/multi-select cards with previews instead of plain text lists
  • Agent Onboarding Guide — Guides new agent creation step-by-step, fills form fields, and navigates the wizard automatically
  • Space Context Query — One-call access to all agents, departments, skills, and providers for informed suggestions
